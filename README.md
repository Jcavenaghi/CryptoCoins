# CryptoInfoApp

CryptoInfoApp es una aplicación web construida con React que te permite explorar información sobre criptomonedas utilizando la API de CoinGecko. La interfaz está estilizada con Tailwind CSS para una experiencia visual atractiva.

## Características

- **Exploración de Criptomonedas:** Ver la lista de criptomonedas junto con sus precios y detalles.
- **Tendencias:** Visualiza las monedas en tendencia actualmente.

## Tecnologías Utilizadas

- **React:** La aplicación está construida utilizando React, una biblioteca de JavaScript para construir interfaces de usuario.
- **CoinGecko API:** Se utiliza la API de CoinGecko para obtener información actualizada sobre criptomonedas.
- **Tailwind CSS:** Los estilos de la aplicación están gestionados con Tailwind CSS, un framework de utilidades de estilos.

## Aprendizajes Obtenidos
- Al desarrollar este proyecto, he aprendido:

- **React**: He adquirido experiencia en la construcción de componentes reutilizables, manejo de estado y enrutamiento con React.
- **Consumo de API**: He aprendido a realizar solicitudes a una API externa (CoinGecko en este caso) para obtener datos en tiempo real.
- **Tailwind CSS**: Me familiaricé con el uso de Tailwind CSS para agilizar el proceso de estilización y crear una interfaz visualmente atractiva.
